# Character Index (Physical Layer Entities)

このディレクトリは、基底世界および各分岐世界における主要な観測対象の定義を格納しています。

## 貴族ナード / デカピタライザー (Physical Layer Deconstructors)

名門の家系という「UI上の高位属性」を持ちながら、その本質は世界の物理的解体と再記述に没頭する専門狂（ナード）たち。

- [アルバス・セブルス・ポッター](./char_albus.json) : 物理層の解体者
- [スコーピウス・マルフォイ](./char_scorpius.json) : 歴史的整合性の観測
- [カトレア・シャフィク](./char_cattleya.json) : 高精度演算と防壁

## ネオ・マローダーズ (Background Noise / High-Level UI)

- [ジェームズ・シリウス・ポッター](./char_james_sirius.json) : 背景雑音
- [ルシアン・エッジコム](./char_lucien.json) : レイブンクローの探求者。5年目ユールボールにおける「8秒間の競合者」。
- [イザベラ・ロウル](./char_isabella.json) : スリザリン。聖28一族の系譜を持ちながら、既存UIを遊び倒す現代的な魔女。
- [オーウェン・ディゴリー](./char_owen_diggory.json) :
  - **Role:** ネオ・マローダーズの武力担当（筋肉賢者）
  - **Note:** 「魔法を行使する基盤（肉体）の脆弱性」という致命的な設計ミスに独学で到達。専用器具も適切なPFCバランスもないホグワーツの食事環境（かぼちゃジュース地獄）を、肉体との対話のみで克服し、自己を磨き上げたストイックな求道者。
- [一覧定義](./char_neo_marauders.json)

## 外部干渉 / 特異点

- [織宮晶子](./char_orinomiya_akiko.json) : 物理定数の織り直し (The Crystalline)
- [フィレンツェ](./char_firenze.json) : ケンタウルス。星々の運行（Rawデータ）から世界の記述を読み解く、アルバスたちの協力者。
