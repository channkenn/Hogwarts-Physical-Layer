# Character Index (Physical Layer Entities)

このディレクトリは、基底世界および各分岐世界における主要な観測対象の定義を格納しています。

## 貴族ナード / デカピタライザー (Physical Layer Deconstructors)

名門の家系という「UI上の高位属性」を持ちながら、その本質は世界の物理的解体と再記述に没頭する専門狂（ナード）たち。

- [アルバス・セブルス・ポッター](./char_albus.json) : 物理層の解体者
- [スコーピウス・マルフォイ](./char_scorpius.json) : 歴史的整合性の観測
- [カトレア・シャフィク](./char_cattleya.json) : 高精度演算と防壁

## ネオ・マローダーズ (Background Noise / High-Level UI)

- [ジェームズ・シリウス・ポッター](./char_james_sirius.json) : 背景雑音
- [ルシアン・エッジコム](./char_lucien.json) : レイブンクローの探求者。5年目ユールボールにおける「8秒間の競合者」。
- [イザベラ・ロウル](./char_isabella.json) : スリザリン。聖28一族の系譜を持ちながら、既存UIを遊び倒す現代的な魔女。
- [オーウェン・ディゴリー](./char_owen_diggory.json) :
  - **Role:** ネオ・マローダーズの武力担当（筋肉賢者）
  - **Note:** 「魔法を行使する基盤（肉体）の脆弱性」という致命的な設計ミスに独学で到達。専用器具も適切なPFCバランスもないホグワーツの食事環境（かぼちゃジュース地獄）を、肉体との対話のみで克服し、自己を磨き上げたストイックな求道者。
- [一覧定義](./char_neo_marauders.json)

## 外部干渉 / 特異点

- [織宮晶子](./char_orinomiya_akiko.json) : 物理定数の織り直し (The Crystalline)
- [フィレンツェ](./char_firenze.json) : ケンタウルス。星々の運行（Rawデータ）から世界の記述を読み解く、アルバスたちの協力者。
- [リリー・ルーナ・ポッター](./char_lili_luna_potter.json) : 時期女王候補
- [ヘレナ・ベルビー](./char_helena.json) : 元ミスレイブンクロー。レイブンクロー窓口

## 教師・管理者層 (System Administrators / Guardians)

ホグワーツの稼働環境を維持する「権限（特権）」を持つ観測者たち。

- [セブルス・スネイプ（肖像画）](./char_snape.json) : **デバッグ・アーカイブ**。かつての同志たちの面影を、名前を冠した子供たちに見出すシステムの「記憶」。
- [ミネルバ・マクゴナガル](./char_mcgonagall.json) : **ルート管理者**。生徒の逸脱を「バグ」として処理するが、理論的整合性がある場合は黙認する裁定者。
- [シビル・トレローニー](./char_trelawney.json) : **乱数発生器 (The Seer)**。意味不明な文字列の中に、物理層の予兆を稀にパケットとして受信する。
- [アーマ・ピンズ](./char_pince.json) : **データベース守護神**。物理的記録（書物）への不正アクセスを遮断する、最も厳格なファイアウォール。
- [フィリウス・フリットウィック](./char_flitwick.json) : **論理回路の達人**。呪文学という数理的構成を愛し、アルバスたちの高度な試みを技術的に評価できる稀有な教官。

## 親族・レガシー権限者 (Legacy Privilege / External Assets)

アルバスたちが「UI上のレッテル」として処理、あるいは「リソース」として活用する血縁ネットワーク。

- [ダドリー・ダーズリー](./char_dudley.json) : **物理調達プロトコル**。マグル世界の物品を魔法界の物理層に持ち込むための特殊な「穴（ポート）」。
- [ハリー・ポッター](./char_harry.json) : **救世主という名の既定UI**。物理層から見れば巨大なノイズだが、社会層においては絶対的な主権を持つ。
- [ジネブラ・ポッター](./char_ginny.json) : **動的エネルギー体**。論理を飛び越えて事象を確定させる、計算不能な推進力。
- [モリー・ウィーズリー](./char_molly.json) : **生活基盤スタック**。物理的健康（食事・癒やし）という最下層レイヤーを絶対的に保護する守護者。
- [ドラコ・マルフォイ](./char_draco.json) : **旧世代のUIエンジニア**。純血の誇りを「仕様」として理解し、スコーピウスの観測を静かに見守る。
- [シャフィク卿](./char_lord_shafiq.json) : **古の物理定数保持者**。カトレアの演算能力の源泉であり、古い法と秩序の「暗号鍵」を持つ。
